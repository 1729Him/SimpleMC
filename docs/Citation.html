Citation¶

We have gathered great codes made by others and put them to work together in the cosmological context. In particular, we use the following:

  • SimpleMC first version written by Anže Slosar and Jose Vazquez.

  • Dynesty developed by Josh Speagle. This contains Ellipsoidal Nested Sampling, Multinest, cubic bounds and ball bounds for nested sampling.

  • MCEvidence by Yabebal Fantaye. This code allows us to estimate bayesian evidence since the chains produced by a Metropolis-Hastings.

  • pyBambi by Will Handley . We use the set of Neural Networks of this code to provide at the Nestle samplers to Neural Networks.

  • Corner by Daniel Foreman-Mackey. This library is for create smooth corner plots.

  • Getdist by Antony Lewis. A package for analysing and plotting Monte Carlo (or other) samples

Cite SimpleMC¶

  • Vazquez, JA., Gómez-Vargas, I., & Slosar, A. (2021). SimpleMC: A package for cosmological parameter estimation and model comparison.

Cite external codes¶

If you consider it pertinent, please also cite:

  • For Ellipsoidal Nested Sampling (snest):

    • Mukherjee, P., Parkinson, D., & Liddle, A. R. (2006). A nested sampling algorithm for cosmological model selection. The Astrophysical Journal Letters, 638(2), L51.

  • For Multinest (mnest):

    • Feroz, F., Hobson, M. P., & Bridges, M. (2009). MultiNest: an efficient and robust Bayesian inference tool for cosmology and particle physics. Monthly Notices of the Royal Astronomical Society, 398(4), 1601-1614.

  • For Artificial Neural Networks in nested sampling:

    • Graff, P., Feroz, F., Hobson, M. P., & Lasenby, A. (2012). BAMBI: blind accelerated multimodal Bayesian inference. Monthly Notices of the Royal Astronomical Society, 421(1), 169-180.

  • For MCEvidence:

    • Heavens, A., Fantaye, Y., Mootoovaloo, A., Eggers, H., Hosenie, Z., Kroon, S., & Sellentin, E. (2017). Marginal Likelihoods from Monte Carlo Markov Chains. arXiv preprint arXiv:1704.03472.

  • If use Corner plots:

    • Foreman-Mackey, D. (2016). corner: Scatterplot matrices in Python. The Journal of Open Source Software, 1.

  • If use Getdist:

    • Lewis, A. (2019). GetDist: a Python package for analysing Monte Carlo samples. arXiv preprint arXiv:1910.13970.
